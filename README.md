# Blood Donation Application

A comprehensive blood donation platform built with React and localStorage for managing blood donation requests, donor search, and funding campaigns.

## Features

### User Management
- User registration with blood group and location
- Role-based access (Admin, Donor, Volunteer)
- User authentication
- User blocking/unblocking (Admin only)
- Profile management

### Donation Requests
- Create and manage donation requests
- View all requests with filtering options
- Filter by blood group, district, urgency, and status
- Request status management (Admin/Volunteer)
- Role-based permissions

### Donor Search
- Search donors by blood group
- Filter by district and upazila (Bangladesh)
- View donor contact information
- Direct call functionality

### Funding Campaigns
- View active funding campaigns
- Make donations to campaigns
- Track funding progress

### Dashboard
- Role-based dashboards
- Statistics and analytics
- Quick actions
- Recent activities
